# Stylized Orc Reference Coverage

Use this reference coverage for the first example asset: `stylized_orc_bruiser`.

## Required Categories

- Heroic stylized full-body proportion references.
- Orc head, brow, jaw, tusk, and ear references.
- Heavy neck, trapezius, shoulder, and upper torso anatomy references.
- Large hand, wrist, and forearm references.
- Hip, knee, ankle, and foot stance references.
- Shoulder armor, bracer, belt, cloth, and leather gear references.
- Skin, leather, cloth, metal, bone, tusk, hair, and eye material references.
- Gameplay-distance readability examples.
- Pose references for melee stance, attack anticipation, and heavy idle.

## Minimum Views

- Front.
- Side.
- 3/4.
- Back if gear affects silhouette.
- Face close-up if facial animation is expected.
- Hand close-up if weapons or grabbing matter.

## Metadata Requirements

Every approved reference should include:

- `domain`
- `body_region`
- `species`
- `style_family`
- `deformation_criticality`
- `target_engine`
- `source_quality`
- `source_path_or_url`
- `license_or_usage_note`
- `what_to_copy`
- `what_to_avoid`

## Blockers

- No style reference.
- No body proportion reference.
- No face or tusk reference for an orc.
- No deformation-zone reference before proportion/blockout approval.
- No material references before material planning.
