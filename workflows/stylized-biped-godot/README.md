# Stylized Biped Godot Workflow

This is the first supported workflow.

## Scope

- Character type: stylized biped.
- Source DCC: Blender.
- Target engine: Godot.
- Default interchange: `GLB` or `glTF`.
- First acceptance example: `examples/stylized_orc_bruiser/`.

## Required Stage Order

Use `docs/pipeline/stage_list.md`.

## Stage History

Use `stage_history.md` to track the full workflow acceptance state against `examples/stylized_orc_bruiser/`.

## Completion Rule

The workflow is complete only when the final asset passes Godot validation and has no hard failures.
