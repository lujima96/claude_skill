# Godot Minimal Import Package Example

This is a tiny `glTF` package fixture for adapter documentation and validator examples. It is not the stylized orc asset.

The package demonstrates the expected file layout:

- `exports/minimal_triangle.gltf`
- `reports/godot_validation_report.md`

Real character validation must still happen inside a Godot project with the target asset, preview scene, and screenshots.
